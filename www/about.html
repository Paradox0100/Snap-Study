About

Snap Study is an app made to study for tests. It allows users to make, study, and share sets. With an intuitive user interface and easy-to-use features, it provides an effective studying experience to maximize your studying. This is a simple web app that allows users to create and share notes. The app is built using HTML, CSS, and JavaScript. The app uses Firebase for authentication and data storage. The app has the following features:

  • Signup and login functionality
  • Create, edit, and delete notes
  • Share notes with other users
  • View notes shared with you
